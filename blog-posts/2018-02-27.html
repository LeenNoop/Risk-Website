Time for a Change

  • Date
    February 27, 2018
  • Posted by
    Asma Bi

Change is a very good thing. It helps you grow and improve. As a group we decided that it was time for us to sit down and make some changes.

Firstly, we thought it would be a good idea to create a plan of action. We created a plan during the first term but we felt it appropriate to create a new one. The plan allowed us to focus on what aspects needed the most improvement. We looked at the group roles again and decided on who would do what. For example, who would be the builders, who would be the testers, the managers etc. Once we had decided on a suitable plan we felt that it was time to book a meeting with the critic.

After meeting with the critic, she gave us advice on how we should list what we should do so that we can make a suitable base game. She showed us how to set our priorities so that we can reach this goal.

It is important that we change how we write code. We need to increase the readability of the code as currently is very difficult to understand. To do this we will have to make sure to include appropriate commenting and follow the general format.

So hopefully after following these changes we can move forward and create a more better and fun game!
