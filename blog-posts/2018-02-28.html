And on the first day...

  • Date
    February 28, 2018
  • Posted by
    Asma Bi & Daniel Sutton

...Team Panda made the intro menu!

The intro menu is the first aspect our users will interact with. The menu instantly immerses the game with music. Getting the players exited and wanting to start playing.

The menu was created using libgdx. There is a Play button which allows the user to begin their conquest. Users will be able to change the volume of in game sound effects and the music in the Options Menu. There is also an exit button which will allow you to exit the game.

A the moment we are using stock images but we have the option to use custom images are being made in-house to making it more personal and real.

This is one step further to finishing the game and it is very exciting to see the project come together finally.
