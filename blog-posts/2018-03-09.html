Testing code, as with any form of testing is very important. You will only know that your code works when you execute it. It is so easy to break the code so it always important to test it as much as possible. Tests can be used so that you can understand how a certain change of code can change and how the system performs. It will let you know if something is wrong.

The current forms of testing we are using include JUnit tests. JUnit allows us to test each of the methods for each class. JUnit increases productivity as it allows us to test as we code. Making sure everything is working. It also reduces the overall amount of time that will be spent on debugging.

Another form of testing that we are using is manual tests. Manual tests make sure the program is doing what it is expected to do. Test cases are executed by someone without any support from tools and scripts. Allowing human observation thus allowing us to make the end result more user-friendly.
