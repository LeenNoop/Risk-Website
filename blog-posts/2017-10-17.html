Now that we have captivated you with our original title and assuming you're here because you're already interested in the product we are making or an Aston University Computer Science assessment marker you can learn more about the progress we make on this project. This project will be an 6 month endeavour to produce a working version of Risk with new features.

There are many things we would like to add to our game. One thing we think players should be able to do is play on a different map than the original by creating their own.

We have created a map builder which takes an image of a map and creates the individual territories players can control to display on the screen. The map must be coloured so each territory has a unique RGB value and then for every pixel in the image the map builder checks its RGB value and creates a territory for each unique colour detected then adding that pixel to the territory, while every preceding pixel with the same RGB value is placed into the territory with its associated RGB value. For each territory a polygon is created and displayed on the screen.

This means a player will be able to create his own map and add the image file to the resource folder and choose it while setting up a new game.
